# DateRangePicker 元件

## 簡介
DateRangePicker 是一個用 React 製作的日期範圍選擇器，支援：
- 點擊設定開始與結束日期
- 點擊較早日期時重設起始日期
- 顯示當前月份
- 禁止點擊非當月日期（呈現灰色，不可點擊）
- 以黃色呈現今天日期

---

## 技術棧
- React 18
- JavaScript
- CSS
- React Testing Library (單元測試)

---

## 專案結構

src/ components/ DateRangePicker.jsx // 主元件 
DateRangePicker.test.jsx // 單元測試 
styles/ DateRangePicker.css // 樣式

---
